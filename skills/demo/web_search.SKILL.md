# Web Search Skill

This skill enables searching the web using a search engine.

## Steps
1. Identify the core information need from the user query.
2. Formulate an effective search query with appropriate keywords.
3. Execute the search using the search engine.
4. Parse the results and extract the most relevant information.

## Constraints
- Search queries should be specific and concise.
- Do not click on suspicious or untrusted links.
- Prioritize recent and authoritative sources.

## Dependencies
- Requires access to a web search API or search engine.

## Examples
- Example 1: {"input": "latest AI research news 2026", "output": "A list of recent AI research articles from arxiv and tech news sites."}
- Example 2: {"input": "Python async best practices", "output": "Top results explaining async/await patterns in Python."}
